### 🚀 Sobre Mim
Sou um desenvolvedor focado em performance, integrações de rede e desenvolvimento para a plataforma Android. Gosto de escavar código em baixo nível para otimizar conexões, criar servidores proxy de alta performance e automatizar tudo que for possível.

- ⚙️ **Foco Atual:** Otimizações de rede I/O, IPC (Inter-Process Communication) no Android e sistemas de streaming de baixa latência.
- 📦 **Projetos Recentes:** Desenvolvi um proxy local em Java com suporte a Range requests e fiz modificações na engine de rede em C++ do PPSSPP.
- 🔧 **Ambiente Preferido:** Linux, Termux e Docker.

### 🏆 Projetos em Destaque

#### 🎮 [PPSSPP Streaming Booster](https://github.com/wendziin/ppsspp_streaming_booster)
> Otimizações de rede I/O em C++ e servidor Node.js adaptativo para streaming de alta performance.
- Modificação na engine do PPSSPP (C++) habilitando **HTTP Keep-Alive** persistente e elevação de prioridade da thread `RemoteIO`.
- Servidor local **Node.js (V17 Adaptive No-Concat)** que gerencia chunks de dados sem gargalos de cópia de memória (`Buffer.concat`), atingindo mais de **170 MB/s** em Wi-Fi 5GHz.
- Integração de configurações e sliders customizados no menu nativo do emulador.

#### 📺 [StreamVault Cast Plugin](https://github.com/wendziin/StreamVaultCastPlugin)
> Plugin Android complementar para IPTV com proxy local integrado e suporte a múltiplos players de transmissão.
- Arquitetura de microsserviço IPC utilizando `Messenger` / `Handler` para integração direta via chamadas RPC.
- Servidor HTTP proxy local multithreaded para resolver restrições de headers do Cast e fornecer suporte completo a Range requests.
- Integração e modificação do aplicativo host **StreamVault-IPTV** com caixa de diálogo para transmissão inteligente via Chromecast ou Web Video Caster (WVC).
