### 🚀 Sobre Mim
Sou um desenvolvedor focado em performance, integrações de rede e desenvolvimento para a plataforma Android. Gosto de analisar código em baixo nível para otimizar conexões de rede, criar servidores proxy locais de alta performance e automatizar ferramentas.

- ⚙️ **Foco de Estudo:** Protocolos de streaming de baixa latência, IPC (Inter-Process Communication) no Android e Engenharia Reversa.
- 📦 **Projetos Principais:** Otimizações na engine I/O em C++ do PPSSPP, servidores proxy locais multithreaded em Java/Node.js e automação de testes.
- 🔧 **Ambiente de Trabalho:** Linux, Termux, Android SDK, Git e Docker.
